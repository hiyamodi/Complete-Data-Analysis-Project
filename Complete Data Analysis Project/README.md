# 📊 E-commerce Sales Analysis Project

A complete data analysis pipeline built with Python to explore sales trends, product performance, and regional market share.

## 📁 Project Structure
- `data/`: Contains the raw `sales_data.csv` file.
- `visualizations/`: Automatically generated charts (`product_sales.png`, `regional_share.png`).
- `report/`: Detailed analysis and business insights.
- `main.py`: The core Python script for data processing.
- `requirements.txt`: List of required Python libraries.

## 🛠️ Technical Requirements
- **Python 3.x**
- **Pandas**: For data cleaning and manipulation.
- **Matplotlib**: For creating data visualizations.
